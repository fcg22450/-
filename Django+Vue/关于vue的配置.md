# Vue配置



1. ### 在电脑中安装node.js

   - ......…..省略

2. ### 使用node.js安装脚手架

   - ```
     npm install -g vue-cli
     ```

3. ### 查看vue版本

   - ```
     vue -V
     ```

4. ### vue下载换源

   - ```python
     npm --registry https://registry.npm.taobao.org
     #临时使用
     ```

   - ```python
     npm config set registry https://registry.npm.taobao.org
     #持久使用
     ```

5. ### 创建一个新的vue工程

   - ```cmd
     vue init webpack 文件名
     ```

6. ### 运行vue工程

   - ```
     vue run dev
     ```
